# Tokens

> Lexical units produced by the lexer and consumed by the parser.

## Categories

### Keywords

```ky
fn, class, final, abstract, enum, contract, type
if, elif, else, while, for, in, match, return
break, continue, defer, guard, unsafe
async, await, static, true, false, none, error, ok
import, from, as, is, and, or, not
```

### Literals

```ky
42 # Integer (i32)
3.14 # Float (f64)
"hello" # String
true / false # Boolean
'a' # Char
none # Option::None
```

### Operators

```ky
+ - * / % # Arithmetic
== != < > <= >= # Comparison
&& || ! # Logical
& | ^ << >> # Bitwise
= := # Assignment and declaration
+= -= *= /= %= # Compound
.. ..= # Range
?? # Null-coalescing
-> # Arrow
as is # Cast and type check
```

### Delimiters

```ky
( ) { } [ ] : ; , . # Parentheses, braces, brackets
```

### Indentation

```ky
: # Indent (block start)
# Indent is a special token generated by the lexer
# based on whitespace changes
```
