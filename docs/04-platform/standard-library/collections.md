# std.collections — List Operations

| Function | Description |
|----------|-------------|
| `sum(list)` | Sum of all elements |
| `product(list)` | Product of all elements |
| `max(list)` | Maximum element |
| `min(list)` | Minimum element |
| `range(n)` | Create list [0, 1, ..., n-1] |
| `range(start, end)` | Create list [start, ..., end-1] |
