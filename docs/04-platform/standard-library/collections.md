# std.collections — List and Array Operations

| Function | Description |
|----------|-------------|
| `sum(lst)` | Sum of all elements |
| `product(lst)` | Product of all elements |
| `max(lst)` | Maximum element |
| `min(lst)` | Minimum element |
| `range(n)` | Create list {0, 1, ..., n-1} |
| `range(start, end)` | Create list {start, ..., end-1} |
