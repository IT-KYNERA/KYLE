# Standard Library

## Modules

| Module | Description |
|--------|-------------|
| `std.core` | Core types: Option, Result |
| `std.collections` | List operations (sum, product, max, min, range) |
| `std.str` | String utilities (startsWith, endsWith, capitalize) |
| `std.io` | File I/O, input, output |
| `std.json` | JSON parsing and stringify |
| `std.math` | Math functions (pow, sqrt, gcd, clamp) |
| `std.time` | Time functions (timestamp, sleep, secondsSince) |
| `std.testing` | Test assertions (assert, assertEq, assertNe) |
