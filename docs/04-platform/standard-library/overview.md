# Standard Library

## Modules

| Module | Description |
|--------|-------------|
| `std.core` | Core types: Option, Result |
| `std.collections` | List operations (sum, product, max, min, range) |
| `std.str` | String utilities (starts_with, ends_with, capitalize) |
| `std.io` | file I/O, input, output |
| `std.json` | JSON parsing and stringify |
| `std.math` | Math functions (pow, sqrt, gcd, clamp) |
| `std.time` | Time functions (timestamp, sleep, seconds_since) |
| `std.testing` | Test assertions (assert, assert_eq, assert_ne) |
