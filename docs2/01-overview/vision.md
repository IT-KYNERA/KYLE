# Vision

Kyle is a compiled, statically-typed programming language designed for building backend systems and command-line tools with native performance, Python readability, and Rust type safety.

## Goals

- **Compilation speed** comparable to Go
- **Runtime performance** comparable to C and Rust
- **Readability** comparable to Python
- **Memory safety** at compile time

## Audience

Kyle is designed for:

- Backend developers building APIs, microservices, and CLIs
- Teams that value code readability as much as performance
- Projects that need native performance without C++ complexity
