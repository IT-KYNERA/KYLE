# Philosophy

## Python readability

Kyle uses indentation for blocks, no semicolons, no braces. Variables are declared without keywords. Code reads like prose.

## Rust type safety

The type system is strong, static, with inference. Everything is known at compile time.

## Go simplicity

Few features, each well-designed. One way to do each thing. No paradigm overload.

## C performance

Compiles to native code via LLVM 18. Optimizations include SSA, inlining, constant folding, and the full LLVM `-O3` pipeline. No runtime overhead, no GC, no implicit refcounting.
